This simple plugin adds a disclaimer that you set up in theme customizer at the end of your posts. It works with all themes.

== Description ==

A simple plugin to add a text of your choice (i.e. a disclaimer) to all your posts. You can use it without tweeking a theme.

== Installation ==

1. Upload the plugin to your plugins directory using your prefered method.
2. Activate the plugin.
3. Setup the disclaimer: a section is added in the Appearance Customizer.
